28 /**
29  * A TCPListener listens for incoming TCP connections and accepts them.
30  * The client should inherit from this and implement the handleNewConnection()
31  * method for when a connection is accepted.
32  *
33  * TCPListeners can be created from any thread, but they will always run on
34  * the thread running the Event::Loop.
35  *
36  * TODO(ongaro): Merge this class into RPC::OpaqueServer. It was originally
37  * separate to hide some libevent internals, but that's no longer the case.
38  */

55  /**
56  * Listen on a new address. You can call this multiple times to listen on
57  * multiple addresses. (But if you call this twice with the same address,
58  * the second time will always throw an error.)
59  * This method is thread-safe.
60  * \param listenAddress
61  * The address to listen on.
62  * \return
63  * An error message if this was not able to listen on the given
64  * address; the empty string otherwise.
65  */
66  std::string bind(const RPC::Address& listenAddress);

74  */
75  virtual void handleNewConnection(int socket) = 0;
