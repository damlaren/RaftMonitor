65 /**
66  * Create a protocol buffer message form a text format.
67  * This is useful for testing.
68  * \tparam ProtoBuf
69  * A derived class of ProtoBuf::Message.
70  * \param str
71  * The string representation of the protocol buffer message.
72  * Fields that are missing will not throw an error, but the resulting
73  * protocol buffer may be less useful.
74  * \return
75  * The parsed protocol buffer.
76  */
