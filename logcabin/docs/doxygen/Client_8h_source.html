178  *
179  * A Tree has a working directory from which all relative paths (those that do
180  * not begin with a '/' are resolved). This allows different applications and
181  * modules to conveniently access their own subtrees -- they can have their own

184  * Methods that can fail come in two flavors. The first flavor returns Result
