24 /**
25  * A File is called by the Event::Loop when a file becomes readable or
26  * writable. The client should inherit from this and implement the
27  * handleFileEvent() method for when the file becomes readable or writable.
28  *
29  * File objects can be created from any thread, but they will always fire on
30  * the thread running the Event::Loop.
31  */

41  * is called first, this File object will close the file descriptor
